## 开发步骤

1. Fork 项目
2. 在项目根目录执行 `npm install`
3. 在项目根目录执行 `eleventy --server`
4. 开发
5. 提交 PR

## 更新文章

在 _games 目录增删改 markdown 文件, 提交, CI 会自动部署
